# 💰 똑딱 (TTokTTak)

<br>

> **" 처음 시작하는 자산관리, 기록을 즐거움으로 바꾸다 "**

TTokTTak은 사용자의 소비 흐름을 직관적으로 기록하고 분석하여 효율적인 소비 습관 형성을 돕는 가계부 서비스입니다.  
자주 사용하는 내역과 반복 결제를 통해 빠르고 간편한 기록 경험을 제공하는 것을 목표로 합니다.

## ⚙️ 실행 방법

### 테스트 계정

- 이메일: jihoon@example.com

### 1. 저장소 클론

```
git clone https://github.com/jeonsungwon9012/Skeleton-project.git
```

### 2. 폴더 이동

```
cd .\Skeleton-project\
```

### 3. npm 설치

```
npm install
```

### 4. 서버 실행

```
# JSON Server 실행 (기본 포트: 3000)
npx json-server db\db.json
```

### 5. 클라이언트 실행

※ 서버 실행 후, 별도의 터미널에서 클라이언트를 실행해주세요.

```
npm run dev
```

## 📑 프로젝트 후기

### 전성원

부족한 부분이 있음에도 불구하고, 팀장으로서 프로젝트의 방향을 설정하고 끝까지 책임지며 팀을 이끄는 경험을 할 수 있었습니다.그 과정에서 팀원들은 흔들림 없이 함께해 주었고, 아낌없는 응원과 격려로 프로젝트의 완성도를 높일 수 있었습니다.

프로젝트를 진행하며 실력, 체력, 시간 등 여러 한계를 마주했지만, 팀원 각자의 강점을 끌어내고 적절히 역할을 배분하며 문제를 해결해 나갔습니다. 그 결과, 서로 다른 역량이 유기적으로 결합되어 하나의 완성된 결과를 만들어낼 수 있었습니다.

특히 개인의 역량을 넘어, 팀원 개개인의 강점을 연결하고 조율하는 과정이 프로젝트의 핵심이었습니다. 협업 구조를 설계하고 이를 유지하는 과정은 쉽지 않았지만, 그만큼 의미 있고 값진 경험이었습니다.

이번 경험을 통해 리더로서 한 단계 성장할 수 있었으며, 함께해 준 팀원들에게 고맙습니다!

### 김대구

프로젝트 초반에는 이해가 부족한 부분이 많아 작은 부분까지 팀원들에게 질문하며 진행했습니다.

하지만 학습을 병행하면서 프로젝트를 진행하니 초기보다 실력이 향상된 것을 체감할 수 있었습니다.

특히 프론트엔드는 거의 처음 접하는 수준이었지만, 팀원들의 도움 덕분에 기능 구현을 완성할 수 있었습니다.

이번 경험을 통해 협업 방식에 대한 이해가 높아졌고, 이후 더 큰 프로젝트를 진행하는 데 도움이 될 것이라 생각합니다.

### 신서영

이번 스켈레톤 프로젝트는 협업의 어려움과 강점을 동시에 느낄 수 있었던 시간이었습니다.

그동안 혼자 개발하는 것에 익숙하다 보니, 팀원들과 기능을 나누고 서로의 코드 스타일을 맞추는 과정이 처음에는 어렵게 다가왔습니다.

하지만 혼자였다면 적당히 타협하거나 놓쳤을 문제들을 팀원들과 공유하며 더 나은 방향을 찾을 수 있었고, 어려운 부분을 함께 공부하는 과정이 특히 든든하고 소중했습니다.

이번에 배운 것들을 토대로 다음 프로젝트에서는 초반 설계부터 꼼꼼히 준비해서, 팀원들과 함께 더 멋진 결과를 만들어보고 싶습니다.

### 최다래

협업에 있어서 소통과 분담이 중요하다는 것을 느꼈습니다. 함께 일관성있는 하나의 서비스를 만들어내기 위해서 각자가 생각하는 걸 하나로 모아 한 방향으로 흐르게 하고, 모르는 것을 다같이 해결해 나가는 과정을 경험해 볼 수 있는 좋은 기회였습니다.

또한 개발에 있어서 협업 프로젝트 경험이 많은 것이 중요하단 것을 느꼈습니다. 협업 프로젝트 경험이 없던 저는 무엇을 해야할 지 바로 잡히지 않아서 무력감을 많이 느꼈기도 했습니다. 더 잘 았으면 더 빠르고 퀄리티 높게 내가 맡은 부분을 해냈을거란 생각을 해서 그래서 더 많이 배우고 경험해보고 싶다는 생각을 했습니다.

팀원들이 잘 못하고 어려워해도 괜찮다고 격려해주시고, 침착하게 잘 이끌어주신 팀장님께도 참 감사했고 다음엔 더 잘하고 싶습니다!
